## Pipeline — Suivi hebdomadaire

### Etape 1 — Revue des metriques (lundi, 15 min)

Se connecter a Google Ads et verifier :

| Metrique | Seuil d'alerte |
|----------|---------------|
| Impressions | < 100/semaine → mots-cles trop restrictifs |
| Clics | < 10/semaine → annonces pas assez attractives |
| CTR | < 3% → titres/descriptions a revoir |
| CPC moyen | > 5 EUR → encheres a ajuster |
| Conversions | Compter formulaires + appels + WhatsApp clics |
| Cout total | Ne pas depasser le budget mensuel fixe |

### Etape 2 — Optimisation des mots-cles (mensuel)

1. **Termes de recherche** : analyser les requetes reelles qui declenchent les annonces
2. **Ajouter** : requetes pertinentes non couvertes en mots-cles
3. **Exclure** : requetes non pertinentes (mots-cles negatifs)
   - Exemples a exclure : "gratuit", "emploi comptable", "formation expert-comptable", "stage"
4. **Ajuster les encheres** : augmenter sur les mots-cles convertissants, baisser sur les non-performants

**Mots-cles cibles** :
- declaration fiscale frontalier luxembourg
- comptable frontalier france luxembourg
- declaration tva luxembourg
- creation entreprise luxembourg
- comptes annuels luxembourg
- comptabilite independant luxembourg
- declaration impots luxembourg frontalier

### Etape 3 — A/B test des annonces (mensuel)

1. Comparer les performances des 3 variantes (CTR, conversions)
2. Pauser la variante la moins performante
3. Creer une nouvelle variante pour tester
4. Respecter la contrainte de 30 chars max par titre, 90 chars max par description

### Etape 4 — Ciblage geographique

Zones cibles prioritaires :
- **Luxembourg** : tout le pays
- **France** : Thionville, Metz, Nancy, Longwy, Forbach (zone frontaliere)
- **Belgique** : Arlon, Aubange (frontaliers BE→LU)

Exclure : reste de la France, Allemagne (sauf zone frontaliere si pertinent)

## Gestion d'erreurs

| Probleme | Action |
|----------|--------|
| Budget epuise avant la fin du mois | Reduire le CPC max, restreindre les horaires (8h-20h), focus mots-cles top |
| CTR tres bas (< 1%) | Revoir les titres/descriptions, verifier l'adequation mots-cles / annonces |
| Beaucoup de clics mais zero conversion | Verifier la page de destination (formulaire fonctionne ? WhatsApp cliquable ?) |
| Cout par conversion trop eleve (> 50 EUR) | Revoir le ciblage, exclure des mots-cles, tester de nouvelles annonces |
| Annonce refusee par Google | Lire le motif, corriger (generalement : claims medicaux, prix trompeurs, marques deposees) |
